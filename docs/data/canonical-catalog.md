# Canonical Catalog — Five-Bag Pilot

**Status:** Draft for adoption · July 2026
**Purpose:** The pilot set for the data-quality experiment: matching, exclusion, condition normalization, variant attribution, aggregation. Expansion beyond these five requires the pilot to meet the quality thresholds in `covetability-score-v0.md` §8.

**Shared global exclusion terms (all bags):** replica, inspired, style of, dupe, look-alike, "type bag", dust bag only, box only, authentication card only, receipt only, strap only, charm, keychain, wallet (unless model IS a wallet), coin purse, mirror only, lock only, key only, repair, "for parts", wanted, WTB, ISO, custom, handmade, doll/miniature/toy, sticker, poster, phone case.

**Gold-set labeling:** 150–200 candidate listings per bag, labeled accept/reject. Rejections use the taxonomy: wrong model · wrong product category · accessory/replacement part · replica or "inspired" · seller misuses model name · child/mini variant mismatched · bundle or lot · wanted-to-buy · indeterminate · duplicate/relist. Accepted listings additionally labeled: variant, size, color family, condition band, completeness (strap/lock/key/dust bag/cards).

## 1. Chloé Paddington

| | |
|---|---|
| **Canonical identity** | Chloé Paddington — Phoebe Philo for Chloé, S/S 2005–~2008 |
| **Aliases** | Paddington Lock Bag, Chloé Padlock Bag, Paddington Satchel, "Chloe paddington" (no accent — expect the unaccented form in most listings) |
| **Variants / sizes** | Medium (canonical satchel), Mini, Bowler, Tote, Zippy; capsule editions |
| **Color families** | Whiskey/Tan (canonical), Black, Chocolate, Cream/Ivory, Grey, Blue, Red, Metallic |
| **Materials** | Grained calfskin; brass padlock hardware (~600g — weight is a common seller talking point and an authenticity heuristic) |
| **Expected asking range** | ~$300 (Fair, incomplete) – $1,800 (Excellent, full set); typical Good/VG $550–950 |
| **Inclusion rules** | Must mention Chloé + Paddington (or alias); padlock reference strongly confirming |
| **Negative keywords (bag-specific)** | "Paddington Bear", "Paddington 2", teddy, plush, DVD, "marmalade" (movie merchandise floods this query), Chloé Edith (distinct model, often cross-listed), Silverado |
| **Common misidentifications** | Edith and Bay listed as Paddington; padlock-style bags from other brands ("paddington style"); replacement padlocks/keys sold alone |
| **Condition concerns** | Leather scratches easily and develops heavy patina; padlock tarnish; missing lock/key crushes value — completeness is a first-class condition input |
| **Authentication risk** | HIGH — among the most replicated mid-2000s bags; superfake density significant |
| **Initial queries** | `chloe paddington bag`, `chloe paddington satchel`, `chloe padlock bag`, category-filtered women's handbags |

## 2. Balenciaga City

| | |
|---|---|
| **Canonical identity** | Balenciaga City (Classic City) — Nicolas Ghesquière motorcycle line, 2001– |
| **Aliases** | Classic City, Moto City, Motorcycle Bag, City Bag; hardware-era terms: Classic (brass), Giant 12/21 (G12/G21), Metallic Edge, Graffiti |
| **Variants / sizes** | City (canonical), Small City, Mini City, First, Town, Work, Weekender, Part-Time — **First/Town/Work/Part-Time are SEPARATE models and must be excluded**, though sellers conflate them constantly |
| **Color families** | Black (canonical), seasonal colors are legion (Rose Gold hw, Anthracite, Rouge, etc.) |
| **Materials** | Distressed lambskin/goatskin ("agneau"/"chèvre" appear in listings) |
| **Expected asking range** | ~$350 (Fair) – $1,600 (Excellent); typical Good/VG $500–900 |
| **Inclusion rules** | Balenciaga + City; require size disambiguation — measurements in the listing beat the title |
| **Negative keywords (bag-specific)** | First, Twiggy, Town, Work, Weekender, Part Time, Velo, Day, Hip, "Le City" ONLY when clearly the 2024 reissue (flag: reissue is a distinct variant, not excluded — see below), Neo Classic |
| **Common misidentifications** | The #1 problem bag for size/model confusion; First mislabeled as City; 2024 "Le City" reissue must be split into its own variant (different market, different price) |
| **Condition concerns** | Corner wear, handle darkening (whipstitch handles absorb oils), mirror missing, tassels split/missing — tassel condition is a seller-photographed staple |
| **Authentication risk** | HIGH — heavily replicated across all eras; "Lampo" zipper and bale details are the community heuristics |
| **Initial queries** | `balenciaga city bag`, `balenciaga classic city`, `balenciaga motorcycle bag city` |

## 3. Fendi Baguette

| | |
|---|---|
| **Canonical identity** | Fendi Baguette — Silvia Venturini Fendi, 1997–; 2019 relaunch ongoing |
| **Aliases** | Zucca Baguette, FF Baguette, Mamma Baguette (WRONG — Mama/Mamma Forever is a separate model; sellers misuse it), Sex and the City bag, 1997 re-edition |
| **Variants / sizes** | Baguette (canonical ~26cm), Mini/Micro Baguette, Baguette 1997 re-edition, embellished/beaded editions (thousands of seasonal variants exist — variant attribution is deliberately coarse: Zucca canvas / leather / embellished / re-edition) |
| **Color families** | Brown Zucca (canonical), Black, beaded/sequin multicolor, seasonal |
| **Materials** | Zucca jacquard canvas, nappa, sequins/beads |
| **Expected asking range** | ~$300 (Fair vintage) – $2,500+ (embellished/Excellent); typical vintage Zucca Good/VG $500–1,100 |
| **Inclusion rules** | Fendi + Baguette; FF-clasp reference confirming |
| **Negative keywords (bag-specific)** | Mama Forever, Mamma (as model), croissant (distinct Fendi model), Peekaboo, Spy, "baguette charm", bread/food listings (yes, really — bakery items match this query) |
| **Common misidentifications** | Mama Forever ↔ Baguette is the dominant confusion (shoulder bags of similar silhouette); vintage vs. 2019+ re-edition must be separated — different price tiers |
| **Condition concerns** | Zucca canvas edge wear, interior sticky lining on some vintage years, bead loss on embellished pieces |
| **Authentication risk** | HIGH for Zucca canvas (widely faked); hologram/serial conventions changed across eras, complicating seller claims |
| **Initial queries** | `fendi baguette bag`, `fendi zucca baguette`, `fendi baguette vintage` |

## 4. Dior Saddle

| | |
|---|---|
| **Canonical identity** | Dior Saddle — John Galliano, S/S 2000; Maria Grazia Chiuri reissue 2018– |
| **Aliases** | Saddle Bag, Dior Oblique Saddle, Galliano Saddle, Trotter Saddle (Trotter = the monogram canvas term sellers use for the era) |
| **Variants / sizes** | Saddle (canonical), Mini Saddle, Saddle with strap (2018+), seasonal prints (Rasta, Girly, Denim, embroidered) — split vintage (2000s Galliano) vs. modern (2018+) as top-level variants: two different markets |

| **Materials** | Oblique jacquard canvas, calfskin, exotic/embellished seasonal |
| **Expected asking range** | vintage: ~$500 (Fair) – $2,500 (Excellent rare print); modern: $2,000–4,500 |
| **Inclusion rules** | Dior + Saddle; 'D' stirrup hardware reference confirming |
| **Negative keywords (bag-specific)** | horse saddle, equestrian, saddle pad, western (equestrian goods flood the query), "saddle style", Gucci/other-brand saddle bags, belt (Saddle belt bags are a separate variant — flag not exclude), card holder |
| **Common misidentifications** | Generic "saddle bag" silhouette from any brand; vintage/modern conflation (price difference is 3–5×) |
| **Condition concerns** | Canvas corner wear, interior Dior-logo lining wear, hardware plating loss on vintage |
| **Authentication risk** | HIGH and rising with the model's revival; vintage date codes inconsistent, weakening seller claims |
| **Initial queries** | `dior saddle bag`, `dior saddle oblique`, `christian dior saddle vintage` |

## 5. Louis Vuitton Pochette Accessoires

| | |
|---|---|
| **Canonical identity** | Louis Vuitton Pochette Accessoires — 1992–; Monogram canvas canonical |
| **Aliases** | Pochette, LV Pochette, Pochette Accessories (misspelling standard), Mini Pochette (SEPARATE model — exclude/split), Pochette NM (2020+ larger re-edition — separate variant) |
| **Variants / sizes** | Pochette Accessoires (canonical ~21cm), Pochette Accessoires NM, Mini Pochette Accessoires (separate), Monogram / Damier Ebene / Damier Azur / Multicolore (Murakami) / Vernis as color-family-level variants |
| **Color families** | Monogram brown (canonical), Damier, Azur, Multicolore white/black, Vernis colors |
| **Materials** | Coated canvas, vachetta leather trim |
| **Expected asking range** | ~$200 (Fair) – $900 (Excellent Monogram); Multicolore commands premium; typical Good/VG $300–550 |
| **Inclusion rules** | Louis Vuitton + Pochette Accessoires (or alias); measurements decisive between Mini / standard / NM |
| **Negative keywords (bag-specific)** | Mini Pochette (when clearly the smaller model), Pochette Métis, Pochette Félicie, Toiletry Pouch, wristlet strap only, extender chain only, insert/organizer (aftermarket accessories dominate this query), Kirigami, wapity |
| **Common misidentifications** | The worst accessory-noise bag of the five: aftermarket straps, chains, inserts, and organizers all keyword-match; Félicie/Métis mislabeled as "pochette"; Mini vs. standard requires measurements |
| **Condition concerns** | Vachetta patina grade is the primary condition driver; interior sticky lining on 1990s pieces; date-code legibility |
| **Authentication risk** | EXTREME — the most counterfeited item class on resale platforms; expect a majority of raw candidate listings to be rejects; this bag is the stress test for exclusion rules |
| **Initial queries** | `louis vuitton pochette accessoires`, `louis vuitton pochette monogram`, `lv pochette accessories` |

## Why these five (recorded rationale)

Deliberately different failure modes: Paddington (movie-merchandise query noise + completeness-driven condition), City (worst size/model conflation + reissue split), Baguette (alias misuse + vintage/re-edition split + seasonal variant explosion), Saddle (cross-domain query noise + 3–5× vintage/modern price split), Pochette (extreme accessory noise + counterfeit density + measurement-dependent matching). All five are high-liquidity models with enough eBay volume to produce meaningful aggregates within 90 days. A matcher, condition normalizer, and aggregation pipeline that survives these five has met every major failure category the 30-bag catalog will contain.
